📊 Sales Overview Dashboard – Power BI
Overview

This Power BI project provides an interactive Sales Overview Dashboard designed to analyze sales performance across different regions, states, and years. The dashboard enables users to monitor key business metrics such as Sales, Profit, Quantity Sold, and Year-over-Year (YoY) growth, helping stakeholders make data-driven decisions.
